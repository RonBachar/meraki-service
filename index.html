03 / המציאות

ניהול רב-אתרי. בדרך הישנה.

ככל שהארגון גדל, כך גדלה המורכבות התפעולית של הרשת. ריבוי ספקים, ציוד מבוזר, וצורך תמידי בנוכחות פיזית הופכים את התחזוקה השוטפת לנטל יקר.

01

פיצול ממשקים

עבודה מול עשרות ספקים וציוד שונה בכל אתר. כל קונסולה. כל ממשק. כל מתודולוגיה.

02

לוגיסטיקה מורכבת

כל עדכון או שינוי תצורה דורש הגעה פיזית של טכנאי לסניף. פריסה של אתר חדש נמשכת ימים.

03

חוסר ויזיביליות

אבחון תקלה מרחוק הופך לאתגר. זמני התגובה איטיים והעלויות מצטברות.

04 / הפתרון

מודל הענן של Meraki.

חומרה, תוכנה ושירותי ענן ביחידה אחת. הפלטפורמה לא רק מחברת מכשירים, היא משנה את אופן ניהול הרשת.

// Featured
90% חיסכון בעלויות התפעול
לעומת ארכיטקטורה מסורתית

פחות שעות איש IT.
פחות נסיעות לשטח.

צמצום דרמטי בזמן הצוות שמושקע בתחזוקה שוטפת, בפריסת אתרים חדשים ובאבחון תקלות מרחוק.

// 01 Zero-Touch

פריסה ללא טכנאי

מוציאים מהקופסה, מחברים לחשמל ולאינטרנט. המכשיר מקבל הגדרות אוטומטית מהענן, ופועל.

// 02 Dashboard

דשבורד אחד מאוחד

ניהול תצורה, ניטור וטיפול בתקלות לכל הסניפים, מאותו מסך. בזמן אמת, מכל מקום.

// 03 Performance

ביצועים ללא פשרות

Web Cache להפחתת שימוש ברוחב פס. QoS ו-SD-WAN לתיעדוף יישומים עסקיים קריטיים.

// 04 Visibility

שקיפות מלאה

אבחון מרחוק, ניטור אוטומטי והתראות. סנכרון תצורה לכל מכשיר מחובר ברחבי הארגון.

05 / הפלטפורמה

מערכת אקולוגית שלמה. נקודת קצה אחת.

פורטפוליו רחב של חומרה ותוכנה, שכולו מנוהל מהדשבורד המרכזי של Meraki.

נתבים אלחוטיים

Routers

קישוריות רציפה לכלל הסניפים, עם ניהול עומסים מרכזי.

מתגי תקשורת

Switches

תשתית קווית עוצמתית עם ניטור פורטים מלא ובקרה מהדשבורד.

רשת אלחוטית

Wi-Fi

נקודות גישה עם אופטימיזציה אוטומטית, אנליטיקה מובנית.

אבטחת רשת

Security

חומת אש מובנית בענן, עם בקרה מאוחדת לכל הארגון.

מערך מצלמות

MV

רכיב משלים בארכיטקטורת הרשת, מנוהל מאותו דשבורד.

06 / הערך העסקי

כשרשת התקשורת הופכת למנוע צמיחה.

Meraki מספקת דאטה עסקי שלא ידעתם שקיים ברשת שלכם. תובנות שנובעות מהתשתית, ומופכות לערך עסקי מדיד.

// OPEN API
API

ממשק פתוח לשילוב נתוני הרשת והמיקום עם CRM ויישומי צד ג', והפיכת הנתונים לפעולות עסקיות בזמן אמת.

// ANALYTICS

אנליטיקה עסקית

ניתוח קצב נוכחות באתר, מדידת זמן שהייה, פילוח מבקרים חדשים מול חוזרים. כלי מדידה אקטיבי להשפעה של פרסום, קידום מכירות ופופולריות מוצר.

// LOCATION

מיקום בזמן אמת

ניתוח מובנה של מיקום, משולב בטכנולוגיית האלחוט. ללא חיישנים נוספים, ללא רכיבים נוספים, ללא תוכנות ייעודיות.

// INTEGRATION

אינטגרציה ארגונית

ממשק פתוח הניתן להתאמה. שילוב נתוני מיקום עם תוכנות צד ג' דוגמת CRM, ויישומים מותאמים אישית למבצעים ממוקדים בזמן אמת.

07 / תהליך הפריסה

חמישה שלבים. מאפיון לפריסה ארצית.

תהליך מסודר שמבטיח התאמה מדויקת לצרכים העסקיים, פריסה מהירה וליווי שוטף לאחר ההפעלה.

01

אפיון

מיפוי מבנה הסניפים, וקטורי התקשורת והצרכים העסקיים.

02

אדריכלות

03

פריסה

שילוח ציוד מוגדר מראש לאתרים. חיבור Zero-Touch מהיר.

04

הגדרה

סנכרון מלא מול הדשבורד המרכזי, החלת מדיניות ארגונית.

05

ליווי

ניטור שוטף, תמיכה וייעוץ להרחבות עתידיות.

08 / Access Manager
// Module 01 / Network Access Control

בקרת גישה לרשת. בענן.

Access Manager הוא שירות תוסף לפלטפורמת Meraki, המאפשר לארגון לנהל בקרת גישה לרשת (NAC) ישירות מתוך הדשבורד הקיים. אכיפה ופיקוח על זהות וגישה מבוססת הקשר, לאנשים ולמכשירים שמתחברים לרשת הארגונית.

  • 01

    אכיפת זהות וגישה

    אכיפה ופיקוח על זהות וגישה מבוססת הקשר. מי, איזה מכשיר, מאיפה, ובאיזה תנאים.

  • 02

    בקרה לאנשים ולמכשירים

    ניהול גישה לכל המשתמשים והמכשירים המתחברים לרשת הארגונית, מנקודת ניהול אחת.

  • 03

    פלטפורמה הניתנת להרחבה

    פלטפורמת אבטחת גישה לרשת המתאימה את עצמה לגודל הארגון, מאתרים קטנים עד רשתות גדולות.

הכל קורה ישירות מלוח המחוונים של Meraki. אותה חוויית ניהול. יכולת מורחבת.

09 / Cisco Spaces
// Module 02 / Smart Spaces

תובנות פיזיות. מתשתית ה-Wi-Fi שלכם.

Cisco Spaces מנצלת את תשתית ה-Wi-Fi של Meraki כדי להפיק תובנות בזמן אמת על תנועת אנשים, נכסים ומכשירים. בעזרת בינה מלאכותית ואנליטיקה מתקדמת, הפלטפורמה מאפשרת לארגונים לנהל את המשאבים שלהם בצורה אפקטיבית, לעמוד ברגולציות, וליצור ערך עסקי ממשי.

// 01 Asset Tracking

ניטור נכסים בזמן אמת

מעקב בזמן אמת אחר נכסים דיגיטליים ותיוג שלהם באמצעות Wi-Fi. כללים למיקום וניצול, התראות על תנועה או שימוש בלתי צפויים.

// 02 Crowd Management

ניהול עומסים במרחבים

טכנולוגיית מיקום פנימי (Indoor Location) למעקב אחרי תנועה, ציוד ונכסים. ניטור עומסים וניהול מיטבי של זמינות חדרים ומשאבים.

// 03 IoT & Compliance

בטיחות ועמידה ברגולציה

ניהול חכם של מלאי מכשירי IoT. מעקב בזמן אמת אחרי מכשירים מחוברים, זיהוי שימוש לא תקין, והפקת דוחות עמידה ברגולציה.

// 04 Experience

חווית עובדים ומבקרים

התראות חזותיות, ניתוב אוטומטי לחדרים וחללים פנויים ברחבי הקמפוס, וצ'ק-אין אוטומטי לנוכחות. חווית משתמש חלקה בכל הארגון.

// 05 Wi-Fi Onboarding

חיבור Wi-Fi קל למבקרים

חוויית התחברות פשוטה וידידותית לאורחים ולמבקרים. תהליכי רישום חכמים ומהירים, וגישה מותאמת לפי סוג המשתמש: אורח, עובד, לקוח.

// Use Case / Real World

בית חולים. בלחיצת כפתור.

בבית חולים גדול, הפלטפורמה מזהה עומסים בחדרי המתנה בזמן אמת ומאתרת ציוד חיוני בלחיצת כפתור. נתוני התשתית הופכים לפעולות שמצילות חיים.

זיהוי עומסים בזמן אמת

מחלקות עמוסות, מסדרונות צפופים, חדרי המתנה. נראות מיידית של כל המתחם.

איתור ציוד חיוני

מכונת הנשמה, עגלת חירום, נייד דיאגנוסטי. מיקום מדויק, מבלי לחפש פיזית.

אינטגרציה במערכות קיימות

התראות אוטומטיות אל צוותים רלוונטיים, באמצעות ה-API הפתוח. נתון הופך לפעולה.

תהליך העבודה של Cisco Spaces.

איסוף, ניתוח, תובנות, פעולה. ארבעה שלבים שהופכים את תשתית ה-Wi-Fi הקיימת למקור אקטיבי של ערך עסקי.

01 / COLLECT

איסוף נתונים

תשתיות Wi-Fi קיימות ומכשירי IoT אוספים נתונים בזמן אמת על תנועה ונכסים.

02 / TRANSFER

העברה לענן

הנתונים מנקודות הגישה ומכשירים מחוברים מועברים לפלטפורמת הענן של Cisco.

03 / ANALYZE

ניתוח בבינה מלאכותית

אלגוריתמי AI מנתחים דפוסי תנועה, מזהים עומסים, ומפיקים תובנות פעילות.

04 / ACT

תובנות ופעולות

הצגה בממשק ידידותי, שליחת התראות אוטומטיות, ואינטגרציה עם מערכות קיימות.
