03 / המציאות

ניהול רב-אתרי. בדרך הישנה.

ככל שהארגון גדל, כך גדלה המורכבות התפעולית של הרשת. ריבוי ספקים, ציוד מבוזר, וצורך תמידי בנוכחות פיזית הופכים את התחזוקה השוטפת לנטל יקר.

01

פיצול ממשקים

עבודה מול עשרות ספקים וציוד שונה בכל אתר. כל קונסולה. כל ממשק. כל מתודולוגיה.

02

לוגיסטיקה מורכבת

כל עדכון או שינוי תצורה דורש הגעה פיזית של טכנאי לסניף. פריסה של אתר חדש נמשכת ימים.

03

חוסר ויזיביליות

אבחון תקלה מרחוק הופך לאתגר. זמני התגובה איטיים והעלויות מצטברות.

04 / הפתרון

מודל הענן של Meraki.

חומרה, תוכנה ושירותי ענן ביחידה אחת. הפלטפורמה לא רק מחברת מכשירים, היא משנה את אופן ניהול הרשת.

// Featured
90% חיסכון בעלויות התפעול
לעומת ארכיטקטורה מסורתית

פחות שעות איש IT.
פחות נסיעות לשטח.

צמצום דרמטי בזמן הצוות שמושקע בתחזוקה שוטפת, בפריסת אתרים חדשים ובאבחון תקלות מרחוק.

// 01 Zero-Touch

פריסה ללא טכנאי

מוציאים מהקופסה, מחברים לחשמל ולאינטרנט. המכשיר מקבל הגדרות אוטומטית מהענן, ופועל.

// 02 Dashboard

דשבורד אחד מאוחד

ניהול תצורה, ניטור וטיפול בתקלות לכל הסניפים, מאותו מסך. בזמן אמת, מכל מקום.

// 03 Performance

ביצועים ללא פשרות

Web Cache להפחתת שימוש ברוחב פס. QoS ו-SD-WAN לתיעדוף יישומים עסקיים קריטיים.

// 04 Visibility

שקיפות מלאה

אבחון מרחוק, ניטור אוטומטי והתראות. סנכרון תצורה לכל מכשיר מחובר ברחבי הארגון.

05 / הפלטפורמה

מערכת אקולוגית שלמה. נקודת קצה אחת.

פורטפוליו רחב של חומרה ותוכנה, שכולו מנוהל מהדשבורד המרכזי של Meraki.

נתבים אלחוטיים

Routers

קישוריות רציפה לכלל הסניפים, עם ניהול עומסים מרכזי.

מתגי תקשורת

Switches

תשתית קווית עוצמתית עם ניטור פורטים מלא ובקרה מהדשבורד.

רשת אלחוטית

Wi-Fi

נקודות גישה עם אופטימיזציה אוטומטית, אנליטיקה מובנית.

אבטחת רשת

Security

חומת אש מובנית בענן, עם בקרה מאוחדת לכל הארגון.

מערך מצלמות

MV

רכיב משלים בארכיטקטורת הרשת, מנוהל מאותו דשבורד.

06 / הערך העסקי

כשרשת התקשורת הופכת למנוע צמיחה.

Meraki מספקת דאטה עסקי שלא ידעתם שקיים ברשת שלכם. תובנות שנובעות מהתשתית, ומופכות לערך עסקי מדיד.

// OPEN API
API

ממשק פתוח לשילוב נתוני הרשת והמיקום עם CRM ויישומי צד ג', והפיכת הנתונים לפעולות עסקיות בזמן אמת.

// ANALYTICS

אנליטיקה עסקית

ניתוח קצב נוכחות באתר, מדידת זמן שהייה, פילוח מבקרים חדשים מול חוזרים. כלי מדידה אקטיבי להשפעה של פרסום, קידום מכירות ופופולריות מוצר.

// LOCATION

מיקום בזמן אמת

ניתוח מובנה של מיקום, משולב בטכנולוגיית האלחוט. ללא חיישנים נוספים, ללא רכיבים נוספים, ללא תוכנות ייעודיות.

// INTEGRATION

אינטגרציה ארגונית

ממשק פתוח הניתן להתאמה. שילוב נתוני מיקום עם תוכנות צד ג' דוגמת CRM, ויישומים מותאמים אישית למבצעים ממוקדים בזמן אמת.

07 / תהליך הפריסה

חמישה שלבים. מאפיון לפריסה ארצית.

תהליך מסודר שמבטיח התאמה מדויקת לצרכים העסקיים, פריסה מהירה וליווי שוטף לאחר ההפעלה.

01

אפיון

מיפוי מבנה הסניפים, וקטורי התקשורת והצרכים העסקיים.

02

אדריכלות

תכנון מבנה הענן, רשתות VLAN ופרופילי האבטחה.

03

פריסה

שילוח ציוד מוגדר מראש לאתרים. חיבור Zero-Touch מהיר.

04

הגדרה

סנכרון מלא מול הדשבורד המרכזי, החלת מדיניות ארגונית.

05

ליווי

ניטור שוטף, תמיכה וייעוץ להרחבות עתידיות.

08 / מודולים מורחבים

שני מודולים נלווים לפלטפורמה.

Meraki Core היא הליבה. שני המודולים האלה משלימים את הפלטפורמה ומרחיבים אותה ליכולות אבטחת גישה ולתובנות מרחב פיזי.

// מודול נלווה: NAC

Access Manager

בקרת גישה לרשת מנוהלת בענן. ניהול זהויות וגישה מבוססת הקשר לאנשים ולמכשירים שמתחברים לרשת הארגונית.

  • אכיפה ופיקוח על זהות וגישה
  • בקרה לפי הקשר: משתמש, מכשיר, מיקום
  • ניהול גישה לאנשים ולמכשירים מחוברים
משולב באותו דשבורד של Meraki. אותה חוויית ניהול, יכולת מורחבת.
// מוצר נלווה: ניהול מתחמים

Cisco Spaces

תובנות עומק מהמרחב הפיזי, על בסיס תשתית ה-Wi-Fi הקיימת. ניטור נכסים, ניהול עומסים, חוויית מבקרים ועמידה ברגולציות IoT.

  • ניטור נכסים בזמן אמת ברחבי הרשת
  • ניהול עומסים ותפוסה במרחבים הארגוניים
  • עמידה ברגולציה דרך ניהול מכשירי IoT
  • חיבור Wi-Fi פשוט ונוח למבקרים ולאורחים
ממנף את תשתית ה-Wi-Fi הקיימת. אין צורך ברכיבי חומרה נוספים.
